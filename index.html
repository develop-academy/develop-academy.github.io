OVERVIEW

One full week of learning.

Students will learn the basics of website development, and by the end of the week will have a customized personal website built from scratch.

The course starts from an introduction to HTML, and teaches:
  • Fundamentals of HTML syntax
  • Essential HTML tags (div, h1-6, p, a, img)
  • More HTML tags (ul, ol, table, etc)
  • HTML5 semantic elements (section, article, header, footer, etc.)
  • Fundamentals of CSS syntax
  • Complex CSS selectors
  • CSS and CSS3 properties (font-size, background, transition, etc.)
  • CSS display model and positioning
  • And more!
